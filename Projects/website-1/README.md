# 🌐 Website UI Project (Horizon Courts)

A modern and responsive **website UI built with React + Tailwind CSS**.
This project focuses on creating a clean layout, structured components, and a visually appealing landing page.

---

## 🌐 Live Demo

👉 https://basic-web-design-1.netlify.app

---

## 🚀 Features

* 🧭 Responsive Navbar
* 🖼️ Hero section with call-to-action
* 📊 Informational sections (About, Stats, Services)
* 🎯 Clean and structured layout
* 🎨 Styled using Tailwind CSS
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Tailwind CSS
* 📦 Vite
* 🧠 JavaScript (ES6+)

## 📸 How It Works

* Navbar provides navigation links
* Hero section highlights the main message
* Multiple sections display content like services, stats, and features
* Layout is built using reusable React components
* Tailwind is used for styling and responsiveness

---

## 🔥 Key Concepts Used

### 1. Component-Based Architecture

* UI is divided into reusable components
* Each section is modular and easy to maintain

---

### 2. Responsive Design

* Built using Tailwind breakpoints
* Works across mobile, tablet, and desktop

---

### 3. Layout Structuring

```js id="g7p7bd"
<div className="max-w-[1400px] mx-auto px-6 py-10">
```

* Centers content with proper spacing

---

## ⚠️ Challenges Faced

### 🔴 Structuring Components

Initially confusing how to divide UI into sections.

✅ Solved by grouping components into:

* section1 (Navbar, Hero)
* section2 (Content sections)

---

### 🔴 Maintaining Consistent Design

Different sections looked inconsistent.

✅ Solved using:

* Common spacing system
* Tailwind utility classes
